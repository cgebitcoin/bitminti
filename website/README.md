## Customization

### Update Links

Replace `YOUR-USERNAME` and `YOUR-HANDLE` with your actual:
- GitHub username
- Twitter/X handle
- Reddit community

### Colors

Edit CSS variables in `style.css`:
```css
:root {
    --primary: #00d4ff;        /* Main accent color */
    --secondary: #ff6b00;      /* Secondary accent */
    --dark: #0a0e27;           /* Background */
    --dark-light: #1a1f3a;     /* Card backgrounds */
}
```

### Content

Edit `index.html` to update:
- Hero text
- Mission statement
- Features
- Network parameters
- Social links
