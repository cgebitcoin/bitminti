Get Started

1

Download

Clone the repository or download pre-built binaries

git clone https://github.com/cgebitcoin/btc3.git
2

Build

Compile from source (or use Docker)

mkdir build && cd build
cmake .. && make -j$(nproc)
3

Run

Start your node and create a wallet

./bin/btc3d -daemon
./bin/btc3-cli createwallet "miner"
4

Mine

Start mining and earn BTC3

./bin/btc3-cli generatetoaddress 1 $(./bin/btc3-cli getnewaddress)
